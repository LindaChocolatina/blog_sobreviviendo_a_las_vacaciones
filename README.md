“Sobreviviendo a las vacaciones” inició como un tarea para vacaciones, con la cual debía explorar el mundo del html y css haciendo una página web narrando los eventos de las vacaciones de fin de año.

Ahora es un blog que aparece en momentos específicos del año, cuando el tiempo cambia de ritmo y la rutina se transforma.
     
Este espacio nace de la experiencia de habitar distintos roles al mismo tiempo, especialmente el de ser mamá perruna, y de intentar equilibrar la vida cotidiana con el deseo de crear, descansar y reconectar.
      
Aquí se comparten fragmentos de esos días: ilustraciones, pensamientos y pequeñas historias que acompañan el ritmo de las vacaciones, con sus desafíos, pausas y momentos inesperados.
      
No es un blog constante, es un refugio temporal.
       
Y quizás, también una forma de recordar que en medio del caos, siempre hay algo que vale la pena observar.
